### Auto Site Data Cleaner
Quickly remove cookies and site data for any website.

> Features:
> 1. Automatically clears site data each time selected sites are opened.
> 2. Save sites in the extension to autoclean them on visit.
> 3. Instantly clear cookies and data for the current site with one click.

### How to install
1. Download this repository and Extract it (Its better to save the extracted folder somewhere safe, so you don't delete it, eg: C Drive)
2. Open Chrome.
3. In the top right, click on the three dots and select:
   - Extensions < Manage Extensions (or open this url: [Chrome Extensions](chrome://extensions/))
4. In the top right corner enable the "Developer Mode".
5. In the left top corner click on "Load Unpacked" Button.
6. Select this folder.
